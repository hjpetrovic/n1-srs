# N1 SRS

Spaced repetition study app for the JLPT N1 exam.

**[Start studying →](https://hjpetrovic.github.io/n1-srs/n1_srs.html)**

## What's inside

- 594 practice questions with verified answers and highlighted target words
- 3,808 N1 vocabulary flashcards with readings and English definitions
- 54 grammar patterns with examples
- SM-2 spaced repetition algorithm
- Google sign-in with cloud sync across devices
- Works offline as a PWA — install it on your phone from the browser

## Setup

The app works immediately in guest mode (progress saves locally). For multi-user cloud sync, see [SETUP.md](SETUP.md).

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `1`–`4` | Select answer / rate card |
| `Space` | Flip flashcard / next card |
| `Enter` | Continue |
